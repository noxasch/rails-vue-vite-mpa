# README

## Prerequisites
- Ruby 3.2.2

## Getting started

```
rails new rails-vue3 -T -J -M -C --skip-jbuilder--css tailwind

bundle add vite_rails
bundle exec vite install
pnpm install

foreman start -f Procfile.dev
```
